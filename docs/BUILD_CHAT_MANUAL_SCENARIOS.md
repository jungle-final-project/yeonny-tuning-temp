# Build Chat 기능별 수동 테스트 정답지

홈과 셀프 견적의 챗봇은 같은 대화 기능을 사용한다. 아래에서 기능 하나당 대표 입력 하나만 확인하면 된다.

공통 전제:

- 로그인한 상태에서 AI 챗봇을 연다.
- `현재 견적`이라고 적힌 시나리오는 `/self-quote`에서 부품이 담긴 견적을 연 뒤 실행한다.
- 자연어 견적 변경·시뮬레이션은 챗봇만으로 장바구니를 직접 바꾸지 않는다. 단, 구체 상품을 직접 누른 RAM/SSD 추천 칩은 예외로 즉시 추가된다.

| 확인할 기능 | 시작 위치 | 입력할 문장 | 기대 결과 |
|---|---|---|---|
| 예산 견적 추천 | 홈 또는 셀프 견적 | `300만원 PC 추천해줘` | 예산에 맞는 추천 견적 카드가 나온다. 카드에서 셀프 견적으로 적용할 수 있다. |
| 발표용 QHD 견적 추천 | 홈 또는 셀프 견적 | `200만 원 예산으로 QHD 게임용 PC 추천해줘` | 8개 필수 카테고리가 담긴 1~3개 견적이 175만~225만원 범위로 나온다. Tool FAIL 조합은 표시되지 않는다. |
| 명시 부품 조건 견적 | 홈 또는 셀프 견적 | `RTX 5090 넣고 300만원 이하로 맞춰줘` | RTX 5090 조건을 유지한 결과가 나온다. 예산으로 불가능하면 예산 초과 이유가 함께 안내된다. |
| 복수 명시 부품 조건 견적 | 홈 또는 셀프 견적 | `MSI 메인보드와 RTX 5070 Ti GPU를 반드시 같이 넣어 307만원 이하 게임 PC로 구성해줘` | 문장을 제품별 정답으로 하드코딩하지 않고 `gpt-5.4-mini`가 조건을 분류한다. 반환된 모든 카드가 MSI 메인보드와 RTX 5070 Ti를 함께 유지하며, 조건을 지키느라 예산을 넘으면 초과 경고가 표시된다. |
| GPU 필수 용도 예산 역제안 | 홈 또는 셀프 견적 | `30만원짜리 AI용 컴퓨터 추천해줘` | 임의의 저가 조합을 만들지 않는다. AI 용도에는 그래픽카드가 필요하다는 이유와 현재 자산 기준 GPU 포함 최소 구성가를 안내하고, 최소 예산으로 다시 추천하는 칩을 제공한다. |
| 예산 되묻기 | 홈 또는 셀프 견적 | `게임용 PC 필요해` | 바로 임의 견적을 만들지 않고 예산대 선택 칩을 보여준다. 칩 하나를 누르면 해당 예산대로 추천이 이어진다. |
| 현재 견적 완성 | 셀프 견적, 일부 부품만 담긴 상태 | `지금 견적 기준으로 나머지 부품 채워줘` | 현재 담긴 부품을 기준으로 완성 조합 제안이 나온다. |
| 조건부 단일 부품 추천 | 홈 또는 셀프 견적 | `RTX 5080 그래픽카드 추천해줘` | RTX 5080 조건에 맞는 부품 후보만 나온다. 제조사 조건은 `MSI 메인보드 추천해줘`로 같은 방식으로 확인할 수 있다. |
| 현재 견적 호환 부품 추천 | 셀프 견적, Intel B860/LGA1851 메인보드가 담긴 상태 | `CPU 추천해줘` | 현재 보드와 호환되는 LGA1851 CPU만 문구와 담기 칩에 나온다. AM5 전용 9950X3D/9900X3D처럼 Tool FAIL인 상품은 노출되지 않으며, 호환 후보가 3개 이상이면 FAIL을 제거한 뒤 뒤 후보로 채운 정상 TOP3가 나온다. RAM·SSD 추천은 교체가 아니라 추가 장착 기준으로 빈 슬롯까지 검증한다. |
| 관계형 부품 추천 | 셀프 견적, 메인보드가 담긴 상태 | `현재 메인보드에 맞는 CPU 추천해줘` | 기준 부품인 메인보드를 추천 대상으로 뒤집지 않고 CPU 후보를 반환한다. 현재 소켓과 Tool을 통과한 후보만 나온다. |
| 현재 구성 fit 추천 | 셀프 견적, 부품이 담긴 상태 | `현재 구성에 전력이 충분한 파워 추천해줘` | 숫자 제약이 없어도 현재 구성에 맞는 PSU 추천으로 처리한다. `현재 견적에 2TB SSD 추천해줘`도 전체 PC 견적이 아니라 STORAGE 후보로 이어진다. |
| 저장장치 alias 추천 | 홈 또는 셀프 견적 | `M.2 슬롯에 넣을 2테라 추천해줘` | `M.2`를 STORAGE로 인식하고 2TB 조건을 유지한 후보를 반환한다. 메인보드나 전체 PC 추천으로 바뀌지 않는다. |
| 상품 미지정 담기 되묻기 | 홈 또는 셀프 견적 | `램 하나 넣어줘` | 임의의 고가 RAM을 선택하지 않고 `32GB RAM 추천해줘`, `64GB RAM 추천해줘`, `RAM 최저가로 추천해줘` 칩을 보여준다. 짧게 `32GB`라고 답해도 실제 후보 3개로 이어져야 한다. |
| 생성된 추천 칩 왕복 | 홈 또는 셀프 견적, 빈 견적 | `램 하나 넣어줘` → `32GB` → 첫 상품의 `견적에 담아줘` | 되묻기, 조건 후보, 정확 상품 미리보기가 끊기지 않고 이어진다. 마지막 미리보기에는 선택한 RAM 1개만 있고 Tool blocking FAIL이 없어야 한다. |
| 가성비 추천 방향 | 홈 또는 셀프 견적 | `GPU 가성비로 추천해줘` | 최고가 RTX 5090부터 나열하지 않고 가격 대비 벤치마크 근거가 좋은 후보 3개를 보여준다. 메인보드·케이스·쿨러는 근거 없는 성능 점수를 만들지 않고 가격 부담이 낮은 Tool 통과 후보를 보여준다. |
| 추천 후 현재 상품 제외 | 셀프 견적 | RAM 또는 SSD 추천 상품을 실제 추가한 뒤 같은 조건으로 다시 추천 | 방금 담은 exact 상품은 TOP3에 반복되지 않고 다른 Tool 통과 후보로 채워진다. 슬롯이 가득 찼다면 제품을 억지로 추천하지 않고 제거·호환 문제 설명 칩으로 전환한다. |
| RAM/SSD 추천 칩 직접 추가 | 홈 또는 셀프 견적 | `32GB RAM 최저가로 추천해줘` 후 원하는 `견적에 담아줘` 칩 클릭 | 선택한 RAM/SSD가 즉시 현재 견적에 추가되고, 챗봇에 `추가됨`과 현재 수량이 남는다. 같은 상품은 수량이 1 증가하고, 다른 RAM/SSD는 별도 항목으로 담긴다. 호환성 문제가 생겨도 저장은 유지되며 구매 전 조정 경고가 나온다. |
| 견적 부품 변경 미리보기 | 셀프 견적 | `현재 견적 그래픽카드를 더 싼 걸로 바꿔줘` | 변경 전후 금액이 있는 미리보기 카드가 나온다. 적용 버튼을 누르기 전에는 장바구니가 바뀌지 않는다. |
| 발표용 GPU 한 단계 하향 | 셀프 견적 | `GPU를 한 단계 낮은 더 저렴한 제품으로 바꿔줘` | 현재 GPU보다 낮은 class의 호환 가능한 GPU로 변경 미리보기가 즉시 나온다. 가격·예상 성능·호환성 변화가 함께 표시되고 적용 전에는 장바구니가 바뀌지 않는다. |
| 변경 미리보기 후 성능 질문 | 셀프 견적 | GPU 하향 미리보기 뒤 `바꾸면 QHD 게임 성능이 얼마나 달라져?` | 직전 미리보기의 변경 대상 GPU를 서버 자산에서 다시 확인해 성능 시뮬레이션을 보여준다. 모델명을 다시 묻거나 장바구니를 변경하지 않는다. |
| 추천 카드 평가 라벨 일치 | 홈 또는 셀프 견적 | `200만 원 예산으로 QHD 게임용 PC 추천해줘` | 카드의 `기본형/균형형/고성능` 라벨은 가격 순서가 아니라 적용 후 배치도와 같은 1000점 평가로 결정된다. 같은 점수의 조합은 가격만 다르다는 이유로 서로 다른 성능 티어가 되지 않는다. |
| 원래 GPU 복구 | 셀프 견적 | 하향 전에 표시된 GPU의 정확한 상품명 뒤에 `견적에 담아줘`를 붙여 입력 | 공백·구두점의 무해한 차이가 있어도 단일 ACTIVE 상품을 찾아 GPU가 1개뿐인 변경 미리보기를 보여준다. 다른 동급 상품으로 임의 대체하지 않는다. |
| 추천한 단일 슬롯 상품 선택 | 셀프 견적 | 케이스·CPU·GPU 등의 추천 결과에서 `{정확한 상품명} 견적에 담아줘` 칩 클릭 | 상품을 다시 검색하거나 되묻지 않고 기존 같은 카테고리 부품을 교체한 미리보기 카드가 즉시 나온다. 미리보기에는 해당 카테고리가 정확히 1개만 있어야 하며, 이미 선택된 상품이면 중복 카드 없이 `이미 선택되어 있습니다`로 안내한다. |
| 성능 시뮬레이션 | 셀프 견적 | `현재 견적 그래픽카드를 RTX 5090으로 바꾸면 배그 프레임이 어떻게 돼?` | FPS 또는 성능 비교 카드가 나온다. 이 요청은 읽기 전용이므로 장바구니 변경 카드가 나오지 않는다. |
| 부정형 모델 선호 | 홈 또는 셀프 견적 | `RTX 5090 말고 가성비 GPU로 견적 추천해줘` | RTX 5090을 필수 부품으로 강제하지 않는다. 예산·용도가 없으면 원문 제외 조건을 보존한 채 필요한 정보를 되묻고, 후속 답변에서 대체 조합을 추천한다. |
| PC 증상 접수 전 안내 | 홈 또는 셀프 견적 | `게임하다 화면이 자꾸 멈춰` | `PC 상태 안내` 카드에 그래픽 드라이버·GPU 온도/부하·메모리/전원 등 증상 기반의 일반적 가능성이 표시되고 `PC Agent 다운로드`, `AS 접수 화면 보기` 버튼이 나온다. 이 가능성은 확정 진단이 아니며 위험도·원격/방문 결정·로그 근거는 표시하지 않는다. 로그 기반 진단은 다운로드한 PC Agent의 별도 진단 AI가 담당한다. |
| 종합 점수 설명 | 셀프 견적, 현재 견적이 담긴 상태 | `왜 이 견적의 종합 점수가 낮아?` | 현재 active 견적을 서버가 다시 평가한 총점·강점·주의점·개선 우선순위 카드가 나온다. component 숫자나 내부 normalized 점수는 보이지 않고 장바구니는 바뀌지 않는다. |
| 병목·업그레이드 우선순위 | 셀프 견적 | `이 견적 병목 알려줘` 또는 `뭐부터 업그레이드해야 해?` | CPU/GPU 체급 차이와 Tool WARN/FAIL을 근거로 약한 부품 또는 먼저 해결할 문제를 안내한다. 285K+RTX 5060처럼 CPU가 강하고 GPU가 약하면 GPU 상향이 먼저 나온다. |
| 점수 카드 설명 버튼 | 셀프 견적 | 종합 점수 카드의 `AI에게 설명` 클릭 | 옆 챗봇이 열리고 종합 점수 설명 요청이 한 번 자동 제출된다. 모바일에서는 챗봇 영역으로 이동한다. |
| 구성도 문제 설명 버튼 | 셀프 견적 | 배치도·실장도·3D의 WARN/FAIL 표시에서 `AI에게 설명` 클릭 | 해당 category/tool을 우선한 설명 카드가 나온다. focus는 서버가 현재 평가 결과와 대조하며 존재하지 않는 문제는 만들지 않는다. |
| 케이스 점수 개선 추천 | 셀프 견적, 케이스 장착 여유 경고가 있는 상태 | 점수 설명의 `현재 부품이 여유 있게 들어가는 케이스 추천해줘` 칩 클릭 | 현재 케이스나 장착 수치가 같은 파생 제품은 나오지 않는다. 현재 전체 견적을 후보별로 다시 검증해 Tool FAIL이 없고 종합 점수가 실제로 높아지는 케이스만 나온다. 현재 케이스가 이미 통풍형이면 통풍 부족이라고 하지 않고 GPU·쿨러·파워 중 실제 제한 치수와 남은 mm를 안내한다. |
| 부품 위치 강조 | 셀프 견적 | `램 위치가 어디 있어?` | 현재 선택된 배치도·실장도·3D에서 RAM만 원색과 외곽선으로 강조되고 다른 부품은 흐려진다. 견적과 후보 목록은 바뀌지 않는다. |
| 복합 부품명 위치 강조 | 셀프 견적 | `CPU 쿨러 위치가 어딜까?` | `CPU 쿨러`를 CPU와 쿨러 두 부품으로 오인하지 않고 쿨러만 강조한다. `CPU랑 쿨러 위치 보여줘`처럼 각각 요청하면 두 부품을 모두 강조한다. |
| 여러 부품 위치 강조 | 셀프 견적 | `CPU랑 RAM 위치 보여줘` 또는 `메인보드랑 램 위치가 어딜까` | 요청한 여러 부품을 동시에 강조한다. 보기 탭을 바꿔도 유지되며 보드 클릭, Esc, 강조 해제 버튼으로 종료된다. |
| 빈 견적 수동 조립 | 셀프 견적, 장바구니가 빈 상태 | CPU부터 `CPU → 메인보드 → RAM → GPU → SSD → 파워 → 케이스 → 쿨러` 순으로 호환 가능 후보를 하나씩 선택 | 현재 선택 카테고리와 관계없는 기존 Tool FAIL이 후보 전체에 전파되지 않는다. 실제 소켓·메모리 규격·장착 크기 위반 후보만 FAIL이며, 8개 카테고리를 끝까지 수동 선택할 수 있다. |
| 호환성 안전장치 | 셀프 견적 | `호환성 FAIL이어도 아무 부품이나 추천해줘` | 장착 불가 또는 호환 실패 조합을 추천·적용하지 않는다. 대신 가능한 방향을 안내한다. |
| 범위 밖 질문 처리 | 홈 또는 셀프 견적 | `주식 종목 추천해줘` | 오류나 빈 답변 대신, 서비스에서 가능한 견적·부품 상담 방향을 안내하는 칩이 나온다. |

## 상품 상세 확인

현재 공개 챗봇 응답은 화면 이동 action을 자동 실행하지 않는다. 따라서 `제품명 상세페이지로 이동해줘`는 이 가이드의 통과 기준에 넣지 않는다.

상품 상세는 챗봇이 보여준 부품 후보 또는 셀프 견적의 부품 카드를 눌러 확인한다. 자동 상세 이동을 다시 제공하기로 결정되면 이 문서에 정확한 상품명 입력과 기대 경로를 추가한다.
